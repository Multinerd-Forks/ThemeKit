  • init(string:textStyle:sizeCategory:textColourStyle:theme:)

    Convenience initialiser for creating an NSAttributedString with ThemeKit styles. Calls self.init(string:attributes:).

    Declaration

    Swift

    public convenience init(string:String, textStyle:TextStyle, sizeCategory:UIContentSizeCategory? = nil, textColourStyle:ColourStyle, theme:Theme? = TKThemeVendor.shared().defaultTheme)

    Parameters

    string

    The text for this attributed string.
    textStyle

    The style that defines the font. Sets the NSFontAttributeName in the attributes dictionary.
    sizeCategory

    The specific UIContentSizeCategory that the font should be created at. nil implies the user’s current UIContentSizeCategory will be used. Default value is nil.
    textColourStyle

    The text colour for this attributed string. Sets the NSForegroundColorAttributeName in the attributes dictionary.
    theme

    The Theme to get the text and colour styles from. Default value is the defaultTheme of the current TKThemeVendor.
